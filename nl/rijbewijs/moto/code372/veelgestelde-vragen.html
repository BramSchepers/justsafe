Code 372

Is code 372 ideaal als beginner?

Voor veel beginners is Code 372 een zeer aantrekkelijke instapstap. Als je al minstens twee jaar in het bezit bent van een rijbewijs B, kun je met slechts 4 uur praktijkles (zonder officieel examen) de toelating krijgen om met een lichte motor of scooter van maximaal 125cc te rijden.

De voordelen voor beginners:

  • Drempelvrij: Je hoeft geen nieuw theorie- of praktijkexamen af te leggen.
  • Snel op weg: Na 4 uur les krijg je het getuigschrift waarmee je de code bij de gemeente kunt laten bijschrijven.

Waar je rekening mee moet houden:

Hoewel het een fantastische start is, heeft Code 372 zijn grenzen. Je bent beperkt tot een vermogen van 11 kW (125cc) en, heel belangrijk: Code 372 is enkel geldig in België. Wil je met de motor de grens over of op een zwaardere machine rijden? Dan is een volwaardig rijbewijs A (A1, A2 of A) noodzakelijk.

Onze conclusie

Zoek je een vlotte manier om te ontdekken of motorrijden iets voor jou is, of wil je simpelweg de file verslaan? Dan is Code 372 de ideale keuze. Wil je echter het echte "tourgevoel" en ook in het buitenland rijden? Dan adviseren we je om direct voor het volledige rijbewijs te gaan.

Welk pakket neem ik best: 2 uur kennismakingsles of direct code 372?

Bij twijfel kun je altijd starten met onze 2 uur durende kennismakingsles motor voor een persoonlijk advies van onze instructeur. Dit is een vrije keuze, je kunt ook direct code 372 volgen. Het is ook mogelijk om de code 372 op een zwaarder type motor te volgen (in samenspraak met de instrcuteur)

Kan ik extra lessen nemen als ik er nog niet klaar voor ben?

Absoluut! Iedereen leert op zijn eigen manier en tempo. Je kunt altijd extra losse lessen boeken in overleg met je instructeur. We bieden ook persoonlijke, doelgerichte rijlessen aan. Na elke les evalueren we samen de stand van zaken en plannen we indien gewenst bijkomende lesuren in.

Zijn er lange wachttijden voor onze code 372 cursus?

Op een mail antwoorden we binnen de 2 werkdagen — meestal nog dezelfde dag. Na je inschrijving heb je binnen de 3 weken je code 372 in handen. Hoe flexibeler je bent (vooral overdag op weekdagen), hoe sneller we je kunnen inplannen.

Wat kost een code 372 in België in totaal?

Bij ons betaal je €395 voor de code 372-rijlessen. Daarbovenop moet je een nieuw rijbewijs laten aanvragen bij je gemeente (ongeveer €25). Dat is verplicht: de code 372 moet op je rijbewijs staan voordat je met een 125cc mag rijden.

Is beschermende kledij verplicht?

Veiligheid staat voorop. Voor de lessen is beschermende motorkledij wettelijk verplicht. Dit omvat een goedgekeurde helm, handschoenen, een motorjas, een lange broek en schoenen die de enkels beschermen.

Geen eigen uitrusting? Geen probleem! Je kunt bij ons gratis gebruikmaken van een helm, jas en handschoenen. Het enige waar je zelf voor moet zorgen is een stevige lange broek (bijv. jeans) en enkelbeschermende schoenen (zoals wandelschoenen of werkschoenen).
