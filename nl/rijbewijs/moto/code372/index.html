Wat is Code 372?

Code 372 is een Belgische aanvulling op je rijbewijs B. Met code 372 mag je tweewielige motoren en scooters besturen tot 125 cc en maximaal 11 kW vermogen. Denk aan:

  • 125cc motorfietsen ‚Äî lichtgewicht motoren
  • 125cc scooters ‚Äî handgeschakeld of automatisch
  • Voertuigen met twee wielen (geen driewielers; daarvoor bestaat code 373)

De regeling geldt alleen op Belgisch grondgebied. Geen volledig motorexamen nodig.

Om de 4 uur 125cc-motorles te volgen en code 372 te verkrijgen, moet je voldoen aan:

  • Je hoeft geen theoretisch of praktisch examen af te leggen bij een examencentrum.
