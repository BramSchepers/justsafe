Code 373

Is code 373 ideaal als beginner?

Code 373 is de weg naar een driewieler (Piaggio MP3, Can-Am, trike) op basis van je autorijbewijs. Ben je minstens 21 jaar en heb je al 2 jaar je rijbewijs B? Dan volstaat een 4 uur durende praktijkopleiding — geen theorie- of praktijkexamen — om legaal met een driewieler in België de baan op te gaan.

Waarom kiezen voor code 373?

  • Geen examen: Er is geen apart theorie- of praktijkexamen; na de opleiding ontvang je het getuigschrift voor de gemeente.
  • Korte opleiding: Met 4 uur les heb je het vereiste attest om code 373 op je rijbewijs te laten bijschrijven.

Beperkingen van code 373

Code 373 geldt uitsluitend voor driewielige motorvoertuigen en enkel op Belgisch grondgebied. Rijden in het buitenland of op een klassieke motor (tweewieler) vereist een volwaardig rijbewijs A (A1, A2 of A).

Conclusie

Wil je een Piaggio MP3, Can-Am of andere trike gebruiken zonder volledig motorrijbewijs? Dan is code 373 de juiste aanvulling op je rijbewijs B. Voor tochten over de grens of op een tweewieler heb je wel een volledige A-opleiding nodig.

Kan ik extra lessen nemen als ik er nog niet klaar voor ben?

Voor code 373 hoef je geen extra les te volgen: er is geen examen, dus de wettelijke opleiding is na 4 uur afgerond. Wij vinden het wel belangrijk dat je je veilig en goed voelt op de driewieler. Wil je voor jezelf geruster zijn of nog wat extra rijervaring opdoen? Dan kun je bij ons altijd extra lessen bijboeken. Dat is volledig vrijblijvend.

Wat is de totaal prijs voor code 373?

De totale kosten voor code 373 zijn €450. Dit omvat de 4 uur praktijkles. Daarna moet je een nieuw rijbewijs aanvragen bij je gemeent en dit kost ongeveer €25.

Wat moet ik meebrengen naar de les?

Neem je identiteitskaart en je rijbewijs B mee — we controleren daarmee of je al minstens 2 jaar in het bezit bent van je rijbewijs. Eigen (beschermende) kledij mag uiteraard; anders lenen wij wat je nodig hebt. Is het bedrag voor de opleiding nog niet overgeschreven? Breng het dan contant mee.

Is beschermende kledij verplicht voor code 373?

Veiligheid staat voorop. Tijdens de driewielerlessen is beschermende motorkledij verplicht. Je hebt een goedgekeurde helm, handschoenen, een motorjas, een stevige lange broek en schoenen die de enkels bedekken nodig.

Geen eigen uitrusting? Geen probleem! Bij ons kun je gratis een helm, jas en handschoenen lenen. Zelf meebrengen: een stevige lange broek (bv. jeans) en schoenen die je enkels beschermen (zoals wandel- of werkschoenen).
